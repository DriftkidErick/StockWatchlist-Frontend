# 📈 StockWatchlist Frontend

A modern React-based frontend for the StockWatchlist application.  
This app allows users to search for stocks, track their performance, and manage a personalized watchlist with real-time market data.

---

## 🚀 Features

- 🔐 User authentication (Login & Register)
- 🔎 Search stocks by name or ticker
- 📊 Auto-fetch real-time stock prices
- ➕ Add stocks to a personal watchlist
- ❌ Prevent duplicate tickers
- 🗑️ Delete stocks from watchlist
- 📈 View price changes and percentage gains/losses
- 🎨 Clean UI using Bootstrap 5

---

## 🛠️ Tech Stack

- React (Vite)
- Axios (API requests)
- React Router (navigation)
- Bootstrap 5 (styling)
